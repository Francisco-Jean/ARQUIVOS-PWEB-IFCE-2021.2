Tabela de Tags HTML Básicas

TAG DESCRIÇÃO
<!DOCTYPE> Representa o tipo de documento e ajuda os navegadores a exibir as páginas da web corretamente. Esse tag deve ser escrita no começo do documento html, antes de qualquer outra tag.
<html> Indica o começo de um documento html. Para indicar o fim do documento, basta usar a tag de fechamento.
<head> Elemento que contem informações que ajudarão o navegador a carregar a página conforme algumas especificações passadas, como estilo da página, descrição e padrões de codificação.
<title> Adiciona um título personalizado para a página, sendo mostrado na barra de título do navegador ou na guia da página.
<body> É a tag que define o corpo principal da página, onde serão adicionados todo o seu conteúdo visual, como textos e imagens.
<h1> Tag que define um texto como um título, podendo ir até o menor tamanho que é definido pela tag h6.
<p> Define um texto como sendo um parágrafo.
