Aprendendo HTML

O Que é HTML?

HTML (HyperText Markup Language - Linguagem de marcação de Hipertexto) é uma linguagem usada para montar páginas web, possibilitando que o programador possa criar cada parte e incluir cada elemento que irá constituir sua página. As páginas criadas podem ser interpretadas por navegadores, de forma que o usuário poderá visualizar o seu conteúdo.

Como surgiu o HTML?

O HTML foi criado em 1991, por Tim Berners-Lee, no CERN (European Council for Nuclear Research) na suíça. Em 1992, foi liberada a biblioteca de desenvolvimento WWW ( World Wide Web), uma rede de alcance mundial, que junto com o HTML proporcionou o uso em escala mundial da WEB.
